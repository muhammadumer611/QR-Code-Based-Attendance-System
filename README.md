# 📖 Overview
The **QR Code Based Attendance System** is an Android application developed to automate the attendance process in educational institutions.
Instead of traditional paper-based attendance, teachers can generate and manage attendance digitally while students can mark attendance using QR codes. The application supports multiple user roles and provides a secure authentication system.
